# Rabbit_QRCode 兔子的炫彩二维码        
本项目是一个二维码生成工具       
使用 `java` 语言，以google的 `zxing` 为基础进行后续开发，想要实现生成一些神奇的二维码 

        本项目不提供成品程序，本来就没太多东西的，想用自己照着代码捣鼓捣鼓就完事了
        简易的食用指南在Main.java中

### 二维码功能
- [x] 普通二维码
- [x] 彩色二维码
- [x] 渐变色二维码
- [x] 二维码中央添加图片
- [ ] 二维码样式
- [ ] 背景图片
